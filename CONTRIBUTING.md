# Contributing to Moneyz

Thanks for contributing.

## Development setup

1. Use Xcode 16.4 or newer.
2. Open `Moneyz.xcodeproj`.
3. Build the `Moneyz` scheme.
4. Run the `MoneyzTests` target before submitting changes.

## Contribution guidelines

- Keep changes focused and small.
- Follow the existing SwiftUI and Swift style already used in the project.
- Add or update tests when changing business logic, view models, repositories, or services.
- Avoid committing personal signing settings, secrets, or generated user data.

## Pull requests

Please include:

- A short summary of the change
- Why the change is needed
- Screenshots or screen recordings for UI changes
- Notes about testing performed
